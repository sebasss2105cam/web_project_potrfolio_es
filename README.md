# Portafolio

Este portafolio muestra un poco lo que he realizado en mis inicios y
aprendizaje como desarrollador web. Se estructuró bajo el modelo e
instrucciones de un brief en Figma, con la información organizada para
diferentes tamaños de dispositivos, de manera que se utilice un diseño
responsivo para la página.

Fue muy interesante entender el uso de Figma para el diseño y posicionamiento
de elementos en las páginas web. Se utilizó HTML5 para la estructura de la página
y CSS3 para el diseño, bajo los parámetros dados en el brief. Fue crucial el uso
de flexbox y media queries para la asignación de reglas de diseño, frente a los
tamaños de dispositivo.
